Технічне завдання: Модуль внесення показань лічильників
1. Мета
Реалізувати окремий модуль API, який дозволяє користувачам:
додавати показання лічильників (наприклад, вода, газ, електрика);
бачити історію показань;
переглядати просту статистику (таблиця + графік).

2. Мінімальний стек
Backend: API на Laravel 12
БД: SQLite, MySQL, PostgreSQL (на вибір).

3. Основна функціональність
3.1. Лічильники
Користувач може мати кілька лічильників різних типів:
Холодна вода
Гаряча вода
Газ
Електроенергія
Тепло
Опціонально: Користувач може додавати зображення лічильників до показань при внесенні.
3.2. Історія
Таблиця показань: дата, значення, різниця з попереднім, фото.
Можливість видалити або відредагувати останній запис.

4. Tenants
Додати можливість додавати адреси і до них користувачів запрошувати.
